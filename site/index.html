Blog

Simple blog created using Laravel. Only admin can add new posts and delte them. On home page recent posts are shown, if there are more than 9, appears button all posts. Posts can be sorted by date they are created. ALso all posts have topics and they can be filtered to show only posts about specific one topic. On single post page it is possible to write comments. All data sent using forms is sent to database. Responsive design created using Bootstrap.

Laravel, PHP, MySQL, Bootstrap, HTML
